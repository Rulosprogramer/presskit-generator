# Presskit Generator

Generador de presskits para artistas independientes con React, Vite y Firebase.

## Configuracion

1. Crea tu archivo `.env` basado en `.env.example`.
2. Completa las variables de Firebase.
3. Para activar el generador de biografia con IA, completa `VITE_GEMINI_API_KEY` con una API key de Gemini.

## IA para Biografia

En el paso 4 del editor (`Biografia`) puedes usar el boton `Generar con IA`.

La IA genera automaticamente:

- `Bio (140 caracteres)`
- `Bio Corta (1 parrafo)`
- `Bio Completa (3-4 parrafos)`

Si falla la generacion, revisa que `VITE_GEMINI_API_KEY` este configurada correctamente.

## Scripts

- `npm run dev`: entorno local
- `npm run build`: build de produccion
- `npm run preview`: vista previa de build

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
